# maglev-robot-control-pipeline
Maglev Microbotics Laboratory project (for MASc thesis: "Automated 3-DoF Control Development for Magnetically-Levitated Microrobots Using Machine Learning").

Currently available is the initial upload of my completed MATLAB + Python algorithms, Simulink control models, and dSPACE ControlDesk UIs for my MASc thesis research on AI-assisted control model development for real-time 3-DoF position control of a magnetically levitating microrobot.

Note that this excludes the raw training data itself, as this data is specific to each individual attractive-type levitation apparatus and would serve no purpose here. The Python algorithms for automatically generating new training data can be found in the `dSPACE_Simulink` folder.
